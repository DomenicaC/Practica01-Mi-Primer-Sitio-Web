CONCLUSIONES:

•	Podemos decir que las páginas web son una gran herramienta ya sea por su capacidad de desarrollo y por las múltiples maneras de realizarlas.

•	Html es un código muy extenso con una amplia gama de oportunidades, así como de desarrollo.

•	Sin duda el desarrollo de las páginas web trae consigo planificación y mejora constante por lo que son abiertas a múltiples opciones.

•	Visual Code es una herramienta intuitiva y de fácil uso por lo que para el desarrollo de páginas web resulta ser de gran ayuda.

•	Las múltiples ventajas que ofrecen las páginas web no tiene limites pues de lo que único que dependemos es de imaginación y desarrollo con herramientas adecuadas.

RECOMENDACIONES:
•	El uso de estos nuevos lenguajes de programación debería ser estudiados y analizados con más propósitos.

•	Al igual que las muchas ventajas podemos decir que incluyen varias restricciones ya sea por navegador web o por código lo que hace que estas herramientas sean muy extensas de comprender.

•	Visual Code ofrece un sin fin de posibilidades, pero sin la correcta guía está herramienta puede volverse tediosa y de difícil uso.

•	La planificación debería constar como paso esencial en el desarrollo de estas páginas web pues cada una de estas debería ser pensada antes de ser desarrollada.
